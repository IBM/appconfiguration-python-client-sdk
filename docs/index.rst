IBM Cloud AppConfiguration Python SDK's documentation! |version|
================================================================

Python client library to quickly get started with the AppConfiguration service

Installation
============
To install, use ``pip`` or ``easy_install``:

.. code:: bash

    $ pip install --upgrade ibm-appconfiguration-python-sdk

or

.. code:: bash

    $ easy_install --upgrade ibm-appconfiguration-python-sdk
